## üîß How It Works

### 1. User visits shop.html
- `cart.js` loads ‚Üí CartManager initialized
- `shop.js` loads ‚Üí Checks if CartManager exists
- 6 products generated from `shopProducts` array
- "Add to Cart" buttons wired up

### 2. User clicks "Add to Cart"
- `cartManager.addToCart(product, quantity)` called
- Product saved to `localStorage.cart`
- Success notification shown
- Cart count updated

### 3. User goes to cart.html
- `cart.js` loads ‚Üí Reads `localStorage.cart`
- Products displayed in cart table
- Quantities, prices, totals calculated

### 4. User refreshes page
- Cart data persists from localStorage
- All items still there


## üêõ If Something Doesn't Work

### Products not showing?
```
1. Open browser DevTools (F12)
2. Check Console tab for errors
3. Verify cart.js and shop.js load
4. Check if cartManager initialized
```

### Add to cart not working?
```
1. Open browser Console (F12)
2. Try: cartManager.addToCart(shopProducts[0], 1)
3. Check: localStorage.getItem('cart')
4. Refresh page and see if saved
```

### Cart page empty?
```
1. Add products on shop.html first
2. Check: localStorage.getItem('cart')
3. Refresh cart.html page
4. Check browser Console for errors
```
