üîå Product API Endpoints

Public - Read Only:

GET /api/products - List all products
GET /api/products/:id - Get specific product

Admin - Full Control (Requires Authentication):

GET /api/admin/products - List all products (with auth)
POST /api/admin/products - Add new product
PUT /api/admin/products/:id - Update product
DELETE /api/admin/products/:id - Delete product

All admin endpoints require an Authorization header with a valid admin JWT token from /api/admin/login
