üîå API Endpoints

Authentication

  • POST /api/auth/register - Register new user
  • POST /api/auth/login - User login

Products

  • GET /api/products - Get all products
  • GET /api/products/:id - Get product by ID

Orders

  • POST /api/orders - Create new order (sends confirmation email)
  • GET /api/orders - Get all orders
  • GET /api/orders/email/:email - Get orders by email
  • DELETE /api/orders/:orderId - Delete order

Admin

  • POST /api/admin/login - Admin authentication
  • GET /api/admin/dashboard - Admin dashboard data

Email

  • POST /api/email/test - Send test email
  • POST /api/email/send-confirmation - Resend order confirmation
  • POST /api/email/send-shipping - Send shipping notification

‚ú® Feature Highlights

üì¶ Order System

Orders are automatically saved when customers checkout. Each order gets a unique ID and timestamp. The system stores customer information, items, totals, and shipping address.

üë®‚Äçüíº Admin Control

Password-protected admin dashboard shows all orders with real-time statistics. Admins can search, filter, view details, export to CSV, and delete orders as needed.

üìß Email Automation

Customers automatically receive order confirmations. Admins can manually send shipping notifications with tracking. All emails use professional HTML templates with brand colors.

üîí Security

Passwords hashed with bcryptjs. JWT tokens for authentication. Admin credentials protected. Email credentials stored in environment variables. CORS enabled.

üì± Responsive Design

All interfaces work perfectly on desktop, tablet, and mobile. Admin dashboard and email templates are optimized for all screen sizes and email clients.

üìä Analytics

Admin dashboard displays total orders, total revenue, revenue this month, and average order value. Real-time statistics help track business performance.
